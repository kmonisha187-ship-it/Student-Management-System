# Student-Management-System
A zero-dependency, single-file Student Management System — dashboard, CRUD, search, CSV export, and localStorage persistence. Just open and use.
Visit here: https://kmonisha187-ship-it.github.io/Student-Management-System/

Student Management System is a clean, fully client-side Student Management System built with vanilla HTML, CSS, and JavaScript. No frameworks, no backend, no dependencies — just open the file and it works.Built as a lightweight, zero-dependency alternative to bloated student portals.

✨ Features

Dashboard — Live stats (total students, departments, average marks, Year 1 count) with a recent enrollments table
Add Student — Form with validation; prevents duplicate Student IDs and emails
View & Filter — Browse all students, filter by name/ID/email, department, or year
Search — Full-text search across name, ID, email, and department with rich result cards
Edit — Update any student record via modal overlay
Delete — Confirmation modal before removing a record
Export CSV — Download the full student list as a .csv file
Persistent Storage — Data saved to localStorage; survives page refresh
Toast Notifications — Success and error feedback on every action
